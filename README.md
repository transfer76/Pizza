## Pizza
Sinatra app for online order pizza

### Description
Language: Ruby, version: 2.5.3

Gem Sinatra

DataBase: SQLite3

### To Start
* Clone **Pizza** repository

* Use bundle
```
$ bundle install
```
* Run migrations
```
$ bundle exec rake db:create
$ bundle exec rake db:migrate
```
* In command line type
```
$ ruby app.rb
```
* Open browser with URL
```
localhost://4567
```
